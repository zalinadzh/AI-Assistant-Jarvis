# 🤖 AI-Assistant-Jarvis

Это мой проект — личный помощник на Python. Я его делала сама, пока учусь на курсе "The Odin Project", просто чтобы разобраться, как работают нейронки.

## 🛠 Как я это запускала

Честно скажу, заставить его работать было тяжелее всего. В России сейчас с этим огромные проблемы. Я несколько дней просто билась с тем, что ключи не работали или сервера были недоступны.

Пришлось перепробовать кучу вариантов, менять настройки сети и искать обходные пути, пока скрипт наконец-то не ответил. Это была самая долгая часть работы — просто пробить стену, чтобы мой компьютер "увидел" сервер.

## 🧠 Что он умеет

Когда связь наконец-то наладилась, я дописала код, чтобы это был не просто чат, а нормальный собеседник:

• Он не забывает: Я сделала так, чтобы он помнил историю нашего разговора. Ты ему пишешь, а он понимает контекст и не тупит.

• Знает меня: Я прописала ему в настройках, что его создала я (Залина). Он знает свою роль и помогает мне по учебе или карьере.

• Не вылетает: Добавила проверки, чтобы если интернет мигнет или сервер лаганет, программа не закрывалась с ошибкой.

## 💻 На чем написано:

Python и Gemini API. Для работы с данными использовала JSON и Requests.



Настроена логика на Python и подключено API OpenRouter.
