# AI-Assistant-Jarvis

Это персональный ассистент на базе Python. Проект создавался вне учебной программы как личная инициатива для освоения работы с API и сложной сетевой инфраструктурой.

## Техническая часть и сложности
Основное время работы над проектом заняло не написание кода, а преодоление инфраструктурных блокировок. Главной задачей было обеспечить стабильное соединение с серверами API из России.

## Что было сделано в процессе:
• Решение проблем с доступом: Пройдено через десятки нерабочих конфигураций, блокировок ключей и сетевых ошибок, пока не был найден стабильный способ обхода региональных ограничений.
• Обработка API: Реализовано прямое взаимодействие с моделью Gemini. Ключи, которые изначально выдавали ошибки, были успешно внедрены в систему после глубокой настройки сетевых запросов.
• Логика контекста: В код внедрена система сохранения истории (chat_history), чтобы ассистент не забывал нить разговора и работал как полноценный собеседник, а не просто чат-бот.
## Функционал
Джарвис настроен под конкретные задачи. В него заложены инструкции, которые определяют его поведение и роли: от технического наставника по веб-разработке до карьерного стратега. Он чётко идентифицирует создателя и понимает свою цель — помогать в обучении и работе.
## Инструменты
• Python
• Google Gemini API
• Библиотеки Requests и JSON для работы с сетью и данными.
